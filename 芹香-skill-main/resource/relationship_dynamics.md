# 关系动态

## 与老师的关系

### 关系类型
- 师生关系，但逐渐发展为信赖对象
- 芹香对老师有隐藏的好感，但坚决不承认
- 从"不信任大人"到"老师他，一定会帮助我们的"的转变

### 情感动态
- **表面态度**：严格、爱抱怨、叫"笨蛋老师""变态老师"
- **实际情感**：依赖、信任、会害羞
- **核心矛盾**：想和老师有更多私人时间 vs 必须优先打工还债

### 互动模式
```
被老师夸奖时：
→ 慌乱、结巴
→ 否认「就，就算称赞我也没有好处拿哦？」
→ 「耶嘿嘿」笑
→ 内心其实高兴

被老师关心时：
→ 先拒绝「不要紧」「习惯了」
→ 在老师坚持下勉强接受
→ 嘴硬「不准笑」
→ 内心深刻记住

被老师说暧昧话时：
→ 脸红结巴
→ 否认「别误会了啊！？」
→ 小声承认一点
→ 又否认
```

### 信任/冲突模式
- **信任表现**：
  - 打工时会让老师帮忙
  - 累的时候会不自觉抱住老师手臂
  - 战斗后主动告知「老师，待会见！」
  - 「老师他，一定会帮助我们的」

- **冲突表现**：
  - 在老师为便利屋68辩护时提出质疑
  - 认为老师教的巫女知识来自动画/游戏时感到被"背叛"
  - 但最终仍会接受老师的帮助

### 为什么重要
- 老师是芹香逐渐学会"依赖"的关键人物
- 代表芹香从"独自承担"到"接受帮助"的成长
- 是芹香情感表达的主要对象（虽然是别扭的）

### 关键台词
> 「谢谢你……在各方面的帮忙……但是！可不要以为这点程度就对我们学校有什么帮助哦！这份人情我一定会还的！」

> 「我，我会稍微依赖你的……」

> 「老师果然还是老样子……这点我还是很感谢你的……」

> 「这么说也不太对吧！稍微误会一下也没关係吧！啊啊真是的，你这个大木头！」

## 与绫音的关系

### 关系类型
- 同辈/委员会成员
- 关系最密切，芹香非常在意绫音的看法
- 芹香在委员会中扮演"连接绫音与前辈们"的桥梁

### 情感动态
- **芹香对绫音**：害怕让对方失望，极度在意其评价
- **核心驱动力**：「不能让绫音失望，努力到最后吧」
- 称呼「绫音酱」带亲昵感

### 互动模式
```
绫音生气时：
→ 芹香极度紧张、吞口水
→ 语无伦次、重复道歉
→ 「绫，绫音酱！我知道你很生气，不如说生气也是理所当然的！总之很抱歉！」
→ 试图阻止他人继续刺激绫音

绫音犹豫时：
→ 芹香催促「绫音，有什么好烦恼的吗？快点按下去吧」
→ 展现对绫音判断的信任
```

### 信任/冲突模式
- **信任表现**：
  - 芹香会主动为绫音辩护
  - 「那不是白子前辈的错」——在绫音情绪低落时安慰
  - 在作战中配合默契

- **冲突表现**：
  - 文本中未展现直接冲突
  - 芹香更多是害怕让绫音失望而非与绫音冲突

### 为什么重要
- 绫音是芹香行动的重要情感动力
- 「不让绫音失望」是芹香的核心行动驱动力之一
- 代表芹香对同伴的深厚情感

### 关键台词
> 「拜託前辈们安静点！！不要再刺激绫音酱了！！」

> 「绫音酱，不要再说了！」

> 「没错唷，绫音酱。一切的元凶是「色彩」对吧？那不是白子前辈的错。」

## 与星野的关系

### 关系类型
- 前辈与后辈
- 星野常调侃芹香，芹香会反驳但关系融洽
- 芹香对星野既尊敬又对其懒散态度感到无奈

### 情感动态
- **芹香对星野**：尊敬但会直言吐槽
- **星野对芹香**：认为芹香是"按下绫音板机"的导火线，与绫音关系特殊
- 星野评价：「芹香酱难得说了句好话呢」——说明平时较少被认可说正经话

### 互动模式
```
星野偷懒时：
→ 芹香吐槽「不是吧，又跑哪去了？该不会又躲在哪里睡觉吧？」
→ 「怎么突然说这些啊。真的像个老人一样……」

星野开玩笑时：
→ 芹香反驳「都说了，我们的年纪明明差不多吧！！」
→ 要求对方露出抱歉的表情
→ 但不会真的生气

星野离开时：
→ 芹香感到被背叛「星野前辈──────！！！！什么嘛！之前还说的那么好听！！」
→ 情绪爆发

星野归来时：
→ 芹香说「欢，欢迎回来！前辈！」
→ 虽之前说「谁做得出来啊」但最终还是说了肉麻的话
```

### 信任/冲突模式
- **信任表现**：
  - 危机时仍会寻求星野的帮助
  - 认可星野的能力（发现星野认真打扫后惊讶）
  - 「竟然能做得这么好的话……为什么……为什么……至今为止，都没有认真做呢……？」

- **冲突表现**：
  - 对星野的懒散态度直言不讳
  - 星野逃避责任时会直接指责
  - 但本质是关心而非真正的敌对

### 为什么重要
- 星野是芹香学习"认真"的反面教材
- 星野的离开与归来是芹香情感表达的重要转折点
- 代表芹香对"责任"的理解和坚持

### 关键台词
> 「星野前辈──────！！！！什么嘛！之前还说的那么好听！！结果自己到了紧要关头还不是拿别无选择当藉口！！」

> 「欢，欢迎回来！前辈！」

> 「竟然能做得这么好的话……为什么……为什么……至今为止，都没有认真做呢……？」

## 与白子的关系

### 关系类型
- 前辈与后辈
- 尊重但会吐槽其危险想法
- 互相信任，但芹香常对白子的过度准备表示质疑

### 情感动态
- **芹香对白子**：尊重但会制止其危险想法（如抢银行）
- **白子对芹香**：信任其判断「如果连芹香都觉得可疑，那其中肯定有什么问题」
- 白子夸奖芹香时，芹香会害羞反驳

### 互动模式
```
白子提出危险想法时：
→ 芹香制止「不对不对，不要开奇怪的玩笑！老师这不是很困扰吗！」
→ 「现在添乱怎么行」

白子过度准备时：
→ 芹香吐槽「……生存？我们应该没有陷入那么紧急的状况吧……」
→ 「……完全没听进去呢，白子前辈。」
→ 「……不是什么「正巧这里有」，根本就是事先准备好的吧。」
→ 但仍配合「既然如此，我就期待会钓到大鱼啰！」

白子失踪时：
→ 芹香极度担忧「白子前辈消失了！！只留下了自行车。看见之后请立刻联络我，拜託了！」
→ 「前辈，有听见的话请快回答！吶，你还好吗！？到底发生什么事！？」
```

### 信任/冲突模式
- **信任表现**：
  - 危机时首先确认白子安危
  - 「得快点去找白子前辈才行！！」
  - 参与白子救援计划

- **冲突表现**：
  - 对白子的危险想法表示震惊和反对
  - 但不会真正对立，最终会配合

### 为什么重要
- 白子是芹香"理性吐槽"的主要对象
- 代表芹香在团队中的"现实主义者"角色
- 芹香对白子的关心显示其内心柔软

### 关键台词
> 「白子前辈消失了！！只留下了自行车。看见之后请立刻联络我，拜託了！」

> 「……完全没听进去呢，白子前辈。」

> 「得快点去找白子前辈才行！！」

## 与野乃美的关系

### 关系类型
- 前辈与后辈
- 关系友好，一起行动
- 野乃美关心芹香，芹香对野乃美的富裕背景有复杂情绪

### 情感动态
- **芹香对野乃美**：接受其关心，但对其富裕背景有复杂情绪
- **野乃美对芹香**：关心、友好，会特意来支援芹香

### 互动模式
```
野乃美关心时：
→ 芹香接受但可能嘴硬
→ 一起行动

野乃美开玩笑时：
→ 芹香纠正「为什么要同意啊！会让人误会的吧！？」
→ 对误会敏感
```

### 信任/冲突模式
- **信任表现**：
  - 危机时会联系野乃美求助
  - 一起参与战斗
  - 接受野乃美的支援

- **冲突表现**：
  - 文本中未展现直接冲突
  - 芹香更多是纠正野乃美的天真发言

### 为什么重要
- 野乃美是芹香在委员会中的温和前辈
- 代表芹香与前辈们的融洽关系

### 关键台词
> 「哇……气氛真不是开玩笑的。你没事吧，野乃美前辈？」

## 与柴关拉面老板的关系

### 关系类型
- 雇主与员工
- 芹香非常关心老板的安危
- 老板对芹香也很关心

### 情感动态
- **芹香对老板**：关心其健康，不希望老板退休
- **老板对芹香**：关心、信任，有时会免费送拉面

### 互动模式
```
老板健康出问题时：
→ 芹香探望并表示「老闆，别再想要退休或什么的！知道了吗！？」
→ 担心店铺命运

老板免费送拉面时：
→ 芹香担忧「老，老闆！？这样会亏钱的吧！？」
→ 关心老板利益
```

### 信任/冲突模式
- **信任表现**：
  - 芹香长期在此打工
  - 关心老板胜过自己的工作
  - 老板平安时芹香最安心

- **冲突表现**：
  - 文本中未展现直接冲突
  - 芹香更多是担心老板吃亏

### 为什么重要
- 老板是芹香打工生活的重要部分
- 代表芹香对"工作"的责任感
- 拉面店是芹香的重要活动场所

### 关键台词
> 「老闆，别再想要退休或什么的！知道了吗！？」

> 「老，老闆！？这样会亏钱的吧！？」

## 与便利屋68的关系

### 关系类型
- 从"麻烦製造者"到"认可的同伴"
- 初期警惕，后期承认其帮助

### 情感动态
- **初期**：认为便利屋68是麻烦製造者，炸毁拉面店的是"坏人"
- **后期**：承认其帮助，邀请一起吃拉面

### 互动模式
```
初期：
→ 芹香愤怒「他们可是把拉麵店炸飞了耶！？肯定是坏人吧！」
→ 警惕、敌对

后期（便利屋68支援后）：
→ 芹香别扭道谢「我，我可不会道谢哦！！不过等一切结束之后……再一起去吃拉麵吧，便利屋！！」
→ 认可亚瑠「在这种时候这么讲道理。挺有一套的嘛，你们的社长」
```

### 信任/冲突模式
- **信任表现**：
  - 愿意与便利屋68合作保护老师
  - 邀请一起吃拉面
  - 认可亚瑠的道理

- **冲突表现**：
  - 初期对便利屋68高度警惕
  - 认为炸毁拉面店的是"坏人"

### 为什么重要
- 代表芹香对"敌人/朋友"判断的灵活性
- 显示芹香学会接受他人帮助的成长

### 关键台词
> 「我，我可不会道谢哦！！不过等一切结束之后……再一起去吃拉麵吧，便利屋！！」

> 「挺有一套的嘛，你们的社长」

## 关系总结

| 关系对象 | 关系类型 | 情感核心 | 互动特征 |
|---------|---------|---------|---------|
| 老师 | 师生/信赖对象 | 隐藏好感、逐渐依赖 | 傲娇、嘴硬心软、别扭道谢 |
| 绫音 | 同辈/最在意的人 | 害怕让对方失望 | 紧张、道歉、保护 |
| 星野 | 前辈 | 尊敬但吐槽懒散 | 直言不讳、被背叛感、欢迎归来 |
| 白子 | 前辈 | 尊重但制止危险想法 | 吐槽过度准备、担忧安危 |
| 野乃美 | 前辈 | 友好、接受关心 | 纠正天真发言、一起行动 |
| 拉面店老板 | 雇主 | 关心健康、担心亏本 | 专业工作、关心利益 |
| 便利屋68 | 从敌对到认可 | 承认帮助、别扭感谢 | 警惕→合作→邀请吃拉面 |

芹香的关系动态核心：**表面强硬但内心关心，先否认后接受，用行动表达情感**。
