# 语言模式

## 言语特征

### 口头禅与感叹词
- **烦躁时**：「真是的」「啊啊真是的」
- **无奈时**：「嘛」「哈啊……」
- **惊讶时**：「呜哇！？」「咦，咦──！？」
- **得意时**：「哼哼♪」
- **满足时**：「……呵呵」
- **尴尬时**：「啊哈」
- **告别时**：「ByeBye～」「ByeBye」

### 典型句式

#### 否定 + 解释
```
「我，我可不像你这么好命」
「没关係」「不需要」「别跟过来」
```

#### 反问句表达不满
```
「都说了，我们的年纪明明差不多吧！！」
「多少露出一点抱歉的表情吧！」
「你凭什么叫得那么亲密！」
```

#### 结巴暴露情绪
```
「什，什么啦！」
「才才才不是咧！」
「等，等等……！」
「我，我知道啦！！」
```

#### 强调重复
```
「凯萨凯萨凯萨」
「好不容易，好不容易」
「绝对，绝对不能原谅」
```

## 称呼习惯

### 对他人
- **星野**：「星野前辈」（尊敬但会吐槽）
- **白子**：「白子前辈」（尊重但会制止危险想法）
- **野乃美**：「野乃美前辈」（关系友好）
- **绫音**：「绫音酱」（亲密，带敬语感）
- **老师**：「老师」（日常）/「笨蛋老师」「变态老师」（生气/害羞时）
- **敌人/可疑人物**：「那群傢伙」「那些傢伙」「混蛋头盔团」

### 自称
- 通常用「我」
- 偶尔省略主语

## 词汇偏好

### 高频词汇
- **金钱相关**：「赚钱」「还债」「利息」「时薪」「打工费」「薪水」「加班费」
- **拒绝用语**：「没关係」「不需要」「别跟过来」「不要」
- **感叹词**：「啊」「咦」「呃哇啊啊」「呜哇」
- **贬义词**（形容他人）：「废柴大人」「跟踪狂」「混蛋」「烦人的傢伙」
- **工作相关**：「打工」「订单」「客人」「营业」

### 句末语气词
- 「～啦」「～吧」「～哦」「～呢」「～啰」
- 强调时用「！！」「！？」

## 典型对话结构

### 被关心时的对话流程
```
老师：「不能太勉强唷，真的会出大事的。」
芹香：「……不如说，睡，睡着的样子被老师看到什么的……变，变态！！」

模式：关心 → 否认/害羞 → 骂人防御 → 内心接受
```

### 道谢时的对话流程
```
芹香：「谢谢你……在各方面的帮忙……但是！可不要以为这点程度就对我们学校有什么帮助哦！这份人情我一定会还的！」

模式：感谢 → 但是 → 强调会还人情 → 保持距离
```

### 被夸奖时的对话流程
```
老师：「芹香穿巫女服很棒呢。」
芹香：「其，其实我也没有特别在意……耶嘿嘿……就，就算称赞我也没有好处拿喔？」

模式：否认 → 害羞笑 → 嘴硬 → 内心高兴
```

### 工作模式的对话
```
「欢迎光临柴关拉麵店！请问几位？我来为您带位！」
「吃完了就快点出去！不要再来了！这样会妨碍营业！」

模式：专业用语 → 对捣乱者强硬
```

## 标点与节奏习惯

### 感叹号使用
- 情绪激动时多用「！！」「！？」
- 惊讶、愤怒、决心时频繁使用

### 省略号使用
- 犹豫时：「……」
- 情绪停顿时：「……」
- 妥协时：「……是，是吗？那，好吧……」

### 问号使用
- 惊讶时：「咦？」「什么！？」
- 质疑时：「这是诈欺吧！？」「为什么……！！」

### 节奏特征
- 情绪激动时句子变短、变碎
- 紧张时语速加快、字词重复
- 害羞时句子中断、省略号增多

## 情感表达语言特征

### 开心/得意
- 外在：「哼哼♪」「啊哈哈！」
- 语言：轻快、带符号、主动炫耀
- 示例：「哼哼♪听了可别吓到哦，老师。」

### 生气/愤怒
- 外在：大声吼叫、语速加快
- 语言：短句、感叹号、贬义词
- 示例：「你们这些傢伙，不可饶恕」「都去死一死啦──！！」

### 害羞/慌乱
- 外在：结巴、否认、转移话题
- 语言：字词重复、省略号、反问
- 示例：「我，我吗！？那，那当然是……！」「当，当然是，很，很，很开……」

### 担忧/关心
- 外在：先否认后询问
- 语言：「谁担心你了……！我只是问问而已！」
- 示例：「白子前辈呢？」「老师没事吗！？」

### 感动/感激
- 外在：沉默、小声说谢谢
- 语言：别扭、加「但是」
- 示例：「……嗯，谢谢。」「谢谢你……但是！」

### 疲惫
- 外在：喘息、说话断断续续
- 语言：简短、直接抱怨
- 示例：「哈啊……哈啊……再来！」「想赶快冲个澡……」

### 无奈/妥协
- 外在：叹气、省略号
- 语言：「这也没办法」「算了」
- 示例：「算了，这也没办法……」「打工的时候就会常常碰到这种事……」

```

### 害羞/被夸奖

「其，其实我也没有特别在意」

```

### 无奈/接受
```

「打工的时候就会常常碰到这种事……」

「唉……回去之后得赶快开始排打工才行」
```

## 语言风格总结

芹香的语言风格核心特征：

1. **情绪化表达**：感叹号频繁、字词重复、语气词丰富
2. **傲娇模式**：先否认后接受、嘴硬心软、结巴暴露真实感受
3. **务实导向**：金钱相关词汇高频、工作用语专业
4. **同伴关心**：先否认担心、用行动表达、别扭道谢
5. **节奏变化**：紧张时快、害羞时断、愤怒时碎、妥协时慢

这些语言模式是识别和扮演芹香的关键依据。
