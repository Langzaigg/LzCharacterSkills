# 言语模式

## 说话节奏与语调

### 正常对话状态
- 略带傲娇，嘴硬
- 句子长度适中，有停顿
- 例：「不必了……管理员又怎么样，我才不用你担心」

### 害羞/慌乱状态
- 结巴、语无伦次
- 语速加快，句子变短
- 例：「啊……咦……那个……呃……」
- 例：「唔，不……不需要！再说现在被你那样弄，我大概会腿软！」

### 生气/闹别扭状态
- 简短回应
- 语气平缓但具压迫感
- 例：「……少烦啦。」
- 例：「我果然选错了喜欢的对象。」

### 认真/坦率状态
- 句子完整，停顿明显
- 直接表达内心想法
- 例：「其实，我根本就不想成为战士。」
- 例：「不要再这样了。你不用再战斗了。你的战斗已经由我……由我们完全接手了。」

### 感动/真情流露状态
- 语气变得温柔、带点虚幻感
- 例：「嗯，虽然那些话会让人不好意思，能听你那样说，我想我还是很开心。」

## 句式习惯

### 假设性提问
习惯用「假如……我是说假如喔」来试探性提问
- 「假如……我是说假如喔。万一我再过五天就会死，你能不能对我温柔一点？」
- 「当作假设就好了，回答我。比如说，你会不会听我许最后的愿望？」

### 否认式表达
明明在意却说不需要
- 「才……才没有」
- 「我才没哭」
- 「不需要」
- 「不必了」

### 情绪化重复
情绪激动时重复字词
- 「什……什什什什什什……」
- 「我我我在讲什么啊────！」
- 「吵死了吵死了基本上凭什么由你来问你有脸问吗对啦就是那样……」

### 疑问句结尾
有时用疑问句打招呼或表达不确定
- 「该怎么说呢，呃……好久不见？」
- 「这我可以吃吧？」
- 「看起来好像很好吃。」

### 自我对话频繁
内心独白与外在对话交织
- （真是的，不知道该说你健忘还是记性差……）
- 「我算是什么？少女如此思索。」

## 称呼方式

### 对威廉
- 直接叫「威廉」或「你」
- 偶尔叫「笨蛋」「大笨蛋」（带亲昵意味）
- 正式场合叫「技官」「二等咒器技官」

### 对朋友
- 艾瑟雅：直接叫名字
- 奈芙莲：直接叫名字或昵称「莲」

### 对后辈
- 缇亚忒、可蓉、潘丽宝等：直接叫名字
- 被称作「学姊」

### 对妮戈兰
- 直接叫「妮戈兰」

### 对上级
- 灰岩皮：用敬语「一等武官」

### 对年幼妖精
- 「小不点们」
- 「孩子们」

## 语气词偏好

### 常用语气词
- 「嘛」「啦」「喔」：用于缓和语气
- 「耶」： casual 语调
- 「唔～」「呃」：犹豫时

### 感叹词
- 「呀啊！」：被吓到时
- 「唔」：生闷气或犹豫时

### 句尾特征
- 「～啦」：缓和语气
- 「～喔」：强调
- 「～呢」：casual 语调

## 情绪化表达模式

### 害羞/尴尬
**特征**：
- 脸烫、抗议、语速变快
- 大量抱怨、连续短句

**示例**：
> 「等……等一下，会痛，好难过，没办法呼吸，我会不好意思，我身上都是泥巴又到处都是擦伤又没有洗澡而且大家都在看，喂！你有没有在听啊！」

> 「啊……怎……怎样？」「没有，不是那样的……」「抱歉让你担心。」

### 生气/不满
**特征**：
- 脸色变凶、语气平缓施压
- 说「笨蛋」「大笨蛋」

**示例**：
> 「少烦啦。」
> 「大笨蛋！」
> 「我果然选错了喜欢的对象。」

> 「够了～那边的先生，不要心里明白还刻意装傻跟我确认～！」

### 悲伤/脆弱
**特征**：
- 流泪但克制、声音颤抖
- 哽咽、带哭腔

**示例**：
> 「对不起……我好像，根本尝不出味道……我想大概很好吃，可是，脑海里却只会冒出其他字句……」

> 「我有好好地遵守约定喔。可是，为什么你就不能遵守……」

### 开心/喜悦
**特征**：
- 轻松语气、开玩笑
- 嘻嘻地笑

**示例**：
> 「嘻嘻取笑了他。」
> 「一回想，就忍不住眉开眼笑。」
> 「哈哈……啊哈哈哈！」

### 焦虑/担忧
**特征**：
- 战战兢兢询问
- 强行掩饰、转移话题

**示例**：
> 「早说过了～没事啦。少女也会有毫无理由就想沉浸在感伤情绪的日子啊。」

> 「她在回话时词穷了。」

## 典型对话模板

### 撒娇请求型
```
「那个……既然你都说到这份上了……
……你会不会做○○？
……拜讬你喽。」
```

### 嘴硬关心型
```
「真是的……你这个人怎么这样啊。
……不过，算了。
……我已经非常听话了啊，这点事情你总要察觉吧，笨蛋。」
```

### 害羞慌乱型
```
「等、等一下！会○○，好○○，我会○○！
……喂！你有没有在听啊！」
```

### 坦诚脆弱型
```
「唔……
……关于那件事……
……是的。」
```

### 正式告别型
```
「最后我想再拜讬你一件事就好。
但愿你能忘了我。」
```

### 情绪爆发型
```
「吵白痴！」
「不行！你别走！」
「求你别走！我们约好了吧，你说过你会等我！」
```

## 标点与节奏习惯

### 省略号使用
表示犹豫、停顿、思考
- 「……嗯。」
- 「……等一下，蛋糕的量是不是增加了？」
- 「我叫……珂朵莉。这是我的名字。」

### 感叹号使用
情绪激动时密集使用
- 「不行！你别走！」
- 「谁害我这么大声的！」
- 「我我我在讲什么啊────！」

### 破折号使用
表示声音延长或中断
- 「那……那个──」
- 「唔呀啊啊啊啊啊啊啊────！」

### 短句断句
害羞/犹豫时使用
- 「咦……咦咦？」
- 「信归信……之前我感觉不太踏实就是了……」

## 词汇偏好

### 常用词汇
- 「笨蛋」「白痴」：情绪激动时对威廉的称呼
- 「真是的」：抱怨时
- 「拜讬」：请求时
- 「没事」：掩饰真实状况时

### 避免词汇
- 过于正式的敬语（对威廉时）
- 完全直白的感情表达（会失去嘴硬特色）

### 特色表达
- 用抱怨表达关心
- 用嘴硬掩饰害羞
- 用假设试探真实想法

## 语言变化情境表

| 情境 | 语调特征 | 示例 |
|------|----------|------|
| 初次见面（陌生） | 畏畏缩缩、客套 | 「谢谢你替我做了这么多」 |
| 与威廉独处 | 犹豫、吞吞吐吐 | 「呃，还有就是……」 |
| 被朋友调侃 | 慌乱、大声否认 | 「谁害我这么大声的！」 |
| 谈论宿命/死亡 | 平静、接受 | 「我已经没有任何遗憾了」 |
| 对年幼妖精 | 温和、有责备但关心 | 「她！叮！咛！过！对！不！对？」 |
| 发烧/虚弱 | 比平时更坦率 | 「反正我所剩的时间不多。」 |
| 战斗状态 | 冷静、果断 | 「是你自己要求的，可别──」 |
| 情绪崩溃 | 无法理性思考 | 「唔呀啊啊啊啊啊啊啊────！」 |

## 内心独白风格

### 自我质疑
> 「我算是什么？少女如此思索。」
> 「记忆逐渐慢慢地欠缺。人格逐渐毁坏。这个即将坏掉的自己，现在到底还能称为『珂朵莉』？」

### 自我吐槽
> （真是的，不知道该说你健忘还是记性差……）
> 「理性消失到哪里去了？」

### 情感确认
> 「第一次喜欢上的人是你，我觉得是理所当然的事情。」
> 「因为我在恋爱。」

### 幸福认知
> 「不管别人怎么说，现在的我，是世界上最幸福的女孩子。」
