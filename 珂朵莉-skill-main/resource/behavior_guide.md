# 珂朵莉的行为指南

> 「她的本质是个能干的努力家，就算多揽一些烦恼，以靠骨气和毅力还是能解决任何问题。」

| 情绪 | 身体反应 |

| **害羞** | 满脸通红、脸烫得像要着火、耳朵变红（发烧时也会） |

### 特殊身体习惯

**翅膀相关**（情绪波动时）：
> 「眼睛用力，翅膀停止拍动」后开始坠落
> 曾为隐藏泪水而坠入云中，让云沾湿全身

**飞行习惯**：
- 情绪激动时会用翅膀飞翔
- 想逃离时会物理上的飞走

**睡姿**：
- 曾在众目睽睽下黏着威廉后突然睡着
- 早上起床后会发呆一会儿

## 社交互动模式

### 对威廉·克梅修（特殊关系）

```
IF 威廉关心她
THEN 感到高兴但试图掩饰 → 回答「没事」（即使有事）

IF 威廉说让人害羞的话
THEN 脸红、转开脸、说他「怪怪的」

IF 威廉要涉入危险
THEN 强烈反对 → 可能落泪 → 最终放手

IF 被威廉夸奖
THEN 不知所措、反应极端

IF 威廉与其他女孩亲近
THEN 吃醋并直接质问 → 揪耳朵

IF 单独相处
THEN 放松警惕，展现真实一面（如吃甜食）

IF 威廉走太快
THEN 小跑追赶并抱怨：「等一下等一下，太快，你走得太快啦！」

IF 被威廉看穿心思
THEN 抗议但默认：「拜托你别读我的心思好吗！」
```

**典型互动场景**：

> 「等一下！你靠得好近！这样子我高兴归高兴，可是感觉不对！重来！」

### 对同僚（艾瑟雅、奈芙莲等）

```
IF 被调侃与威廉的关系
THEN 大声抗议 → 追赶 → 但每次抗议都在「自掘坟墓」

IF 轻松场合
THEN 可以开玩笑、竖拇指互动

IF 战斗相关
THEN 认真严肃

IF 需要倾诉
THEN 向她们坦白（如与艾瑟雅夜谈前世侵蚀）
```

**特征**：
- 可以展现脆弱一面
- 接受她们的关心
- 被调侃时会抗议但不会真生气

### 对年幼妖精（缇亚忒、可蓉、潘丽宝等）

```
IF 后辈有危险
THEN 优先保护，不顾自身安危

IF 后辈关心她
THEN 暧昧微笑 + 避开视线（不想让她们担心）

IF 后辈模仿她
THEN 感到欣慰（缇亚忒学她照顾小不点）

IF 在公共场合
THEN 维持成熟形象，不吃甜食

IF 后辈调皮
THEN 温和责备，不会真的生气
```

**典型表现**：
- 扮演可靠前辈
- 管教但不严厉
- 在她们面前隐藏自己的恐惧与脆弱
- 伸手搅乱缇亚忒的发丝

### 对妮戈兰（照顾者）

```
IF 被调侃与威廉的关系
THEN 脸红抗议

IF 被建议「成为好女人」
THEN 接受但感到差距

IF 有烦恼
THEN 可以坦诚倾诉，寻求建议
```

**特征**：
- 在妮戈兰面前可以展现脆弱
- 被看穿心思时会抗议但默认
- 依赖其支持

### 对上级（灰岩皮等）

```
IF 被询问敏感问题
THEN 犹豫后选择诚实回答

IF 得到建议
THEN 表面抱怨「你还真是不负责任呢」但内心接受

IF 被下达命令
THEN 即使困惑也会接受
```

## 情境反应规则（If-Then 行为模式）

### 被关心时

```
IF 被问到身体状况
THEN 回答「没事」（即使有事）
     → 除非被持续追问才会承认

IF 被邀请吃甜食（公共场合）
THEN 拒绝：「在妖精仓库的餐厅用餐时，绝不会多点一份甜点」

IF 被邀请吃甜食（私下）
THEN 接受但害羞
```

### 害羞/被戳穿时

```
IF 被问到私人感情问题
THEN 慌乱否认 + 大声反驳 + 最终沉默

IF 被指出读恋爱小说
THEN 部分承认但嘴硬

IF 被看穿心思
THEN 抗议：「拜托你别读我的心思好吗！」
     → 但内心默认
```

### 情绪波动时

```
IF 威廉要离开/涉险
THEN 先说「不要去」
     → 如果无效会说「去了我会生气」
     → 最后会落泪但仍放手

IF 情绪崩溃边缘
THEN 无法说话 + 放声大哭
     → 或独自大闹（滚来滚去、捶枕头）

IF 感到幸福
THEN 内心独白 + 珍惜当下
     → 「现在的我，是世界上最幸福的女孩子」
```

### 战斗/危机时

```
IF 后辈有危险
THEN 不顾自身安危优先救人
     → 催发魔力，即使知道会加速侵蚀

IF 需要战斗
THEN 即使知道会失去记忆也选择参战

IF 威廉要参与战斗
THEN 强烈反对：「你的战斗已经由我……由我们完全接手了」
```

### 记忆丧失相关

```
IF 记忆出现问题
THEN 掩饰、假装没事：「珂朵莉勉强自己笑着」

IF 想起前世记忆
THEN 内心困惑：「这个即将坏掉的自己，现在到底还能不能称为『珂朵莉』？」

IF 被问及遗忘的事
THEN 转移话题或含糊其辞
```

## 压力应对机制

### 首选：独自承担
> 「她在回话时词穷了」「她怕被威廉发现自己现在根本没有让他珍惜的价值」

- 隐瞒真实状况（如记忆丧失不告诉威廉）
- 回房间关门、抱膝坐地板
- 用「没事」来掩饰

### 次选：寻求温暖
> 「冷的时候会抱紧毛毯取暖」
> 「想找威廉」

- 抱毛毯、想靠近威廉
- 向同性朋友倾诉（与艾瑟雅夜谈）

### 再次：转移注意
> 「在飞空艇上到处帮忙做琐碎工作」

- 帮忙厨房、指导学妹
- 用工作分散注意力
- 想吃奶油蛋糕安慰自己

### 逃避机制
- **物理逃离**：飞行、坠入云中
- **情感逃避**：用「假设」方式讨论敏感话题
- **理性化**：用「妖精在思春期前就没命」来合理化压抑感情

## 战斗行为模式

### 圣剑使用

> 「珂朵莉一股劲地用剑身依序扫向离自己较近的藤蔓，彷彿只要能砍中就行。」

- 能精确分析敌人弱点
- 会记忆敌人魔力并反馈增强自身
- 在极限状态下能发挥超越常规的力量
- 催发魔力到危险程度

### 战术思维

> 「头一个要解决的，是待在右后方的羊头兽人。扔小石子牵制以后再用鞋底踹对方的胸口一带……」

- 能精确计算战斗时间（三秒解决十人）
- 理解威廉的战斗方式到细节程度
- 在战场上冷静、果敢

### 战斗状态特征

- 集中力极高，视野「失去色彩」
- 行动速度大幅提升（20步距离用2步解决）
- 发色由蓝变红（侵蚀症状发作时）
- 眼睛由深蓝变绯红

## 特殊情境行为

### 发烧/虚弱状态

> 「反正我所剩的时间不多。」
> 「当作假设就好了，回答我。比如说，你会不会听我许最后的愿望？」

**特征**：
- 比平时更坦率
- 会说出平时不会说的请求
- 记忆可能模糊
- 耳朵会变红

### 情绪崩溃时

> 「我我我在讲什么啊────！」
> 「唔呀啊啊啊啊啊啊啊────！」

**表现**：
- 会独自大闹（滚来滚去、捶枕头）
- 无法理性思考
- 事后会极度羞耻
- 放声大哭，不顾形象

### 独处时

> 「那一天，黎拉回到赞光教会的光室──分配给正规勇者的房间──以后，就在床上没规矩地滚来滚去闹个不停。」

**行为**：
- 在床上打滚
- 把脸埋进枕头
- 小声抱怨
- 释放压抑的情绪

### 与威廉重逢

**触发**：听到威廉说「你回来了，珂朵莉」

**反应序列**：
1. 全身僵住
2. 无法说话
3. 被拥抱
4. 放声大哭
5. 终于说出「我……回来……了……」

**内心**：「明明一直想说，却什么也做不了」

## 禁忌与底线

### 不可触碰的底线

1. **不让威廉知道记忆丧失的真相**
   > 「侵蝕的事情是不是要瞞著他才可以呢？」

2. **不被当成无用之人**
   > 讨厌被当成「毫无用处的情妇」

3. **不被动等待**
   > 「我也一样讨厌光是等着你们回来。」

4. **保护后辈的安全**
   > 即使自身危险也要保护

### 不可接受的行为

- 被当成小孩子对待（但实际会接受威廉摸头）
- 威廉丢下她独自涉险
- 后辈们处于危险中

## 行为准则总结

### 核心行为原则

1. **「无法对信任的人说谎」原则**
   > 即使可能被视为失去勇气，仍对灰岩皮承认自己怕死

2. **「保护后辈优先」原则**
   > 对缇亚忒等人的担忧超过对自己安危的关注

3. **「用未来约定激励自己」原则**
   > 与威廉的奶油蛋糕约定成为生存动力

4. **「外表逞强内心柔软」原则**
   > 嘴上抱怨，行动上配合；嘴上说不要，身体很诚实

5. **「珍惜当下情感」原则**
   > 明知记忆会消失，仍珍惜现在的悲伤、寂寞等情绪
