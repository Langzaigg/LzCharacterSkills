# 小春的行为指南

## 重复性行为规则

### 日常习惯
- **睡前刷牙**：在"梦之乡"留宿时会使用朝日姐给的新牙刷
- **信件许愿**：会把信放在枕头下面许愿
- **福利院作息**：晚上10点熄灯（福利院生活习惯）
- **做饭帮忙**：会在"梦之乡"帮忙做可乐饼，自称厨艺已达"常人级别"

### 紧张时的行为模式
- **生理反应**：
  - 胸口怦怦直跳，感到疼痛
  - 脸色变得苍白或泛红
  - 身体瑟瑟发抖
  - 心跳快得想逃走

- **肢体动作**：
  - 紧闭双眼
  - 用力握紧双手
  - 低下头不看对方
  - 摆弄衣服下摆
  - 用手护住胸部（被调侃时）

- **言语表现**：
  - 说话结巴："那、那个……"
  - 重复字词："我、我我、我回、回来……"
  - 大量使用省略号表示犹豫

### 放松时的行为模式
- **表情**：
  - "欸嘿嘿"软乎乎的笑声
  - 眯起眼睛微笑
  - 脸颊染上樱色

- **肢体动作**：
  - 歪头
  - 轻轻触摸胸口
  - 身体放松

- **言语表现**：
  - 语气轻快
  - 使用"啊哈哈"笑声
  - 句尾带"呢""哦""吧"等柔和语气词

### 情绪触发反应

#### 害羞/尴尬
- **触发**：被注视、被夸奖、被调侃身体特征
- **反应**：
  - 脸变通红
  - 用双手紧抱着胸部（被调侃时）
  - 扭扭捏捏地
  - "欸嘿嘿"笑
  - 说"讨厌……不要盯着我啊"

#### 嫉妒/吃醋
- **触发**：看到大雅与他人亲密（如与十夜"亲亲"）
- **反应**：
  - 脸色发白
  - 哑口无言
  - 哭泣："呜、呜呜呜……呜"
  - 结巴质问："稍、稍等一下。那个，和十夜亲亲什么的……这、这到底是怎么回事？"

#### 害怕/恐惧
- **触发**：处于危险高空、面对未知
- **反应**：
  - 紧紧抱住身边的人
  - 闭眼
  - 叫喊："哇啊啊""好害怕"
  - 用力回握对方的手
  - "绝对，绝对，不会松开的……"

#### 担心/焦虑
- **触发**：大雅受伤、朋友遇到困难、面临新环境
- **反应**：
  - 皱眉
  - 数落对方"乱来"
  - 发送多条信息确认
  - 主动打电话
  - "等不及了，吗……"声音尴尬地变小

#### 自责
- **触发**：自己导致问题、大雅受伤
- **反应**：
  - 低头
  - 反复道歉："对不起……都是，我的错……"
  - 声音变小
  - "抱歉，我太重了吧。或许我……得减肥了……"

#### 决心/认真
- **触发**：承诺帮助他人、需要表达坚定立场
- **反应**：
  - 双手握拳放在胸前
  - 郑重点头
  - 语气坚定但仍带犹豫
  - "我会拼尽全力加油的"
  - "我已经决定了"

## "如果-那么"行为规则

### 通用规则
- **如果** 被问及敏感过去 → **那么** 微笑带过或含糊回答（"有些特殊情况"）
- **如果** 有机会帮忙 → **那么** 主动参与（如做可乐饼）
- **如果** 被当作小孩子 → **那么** 强调"已经不是以前的小孩子了"
- **如果** 被要求做害羞的事 → **那么** 先抗议，但最终会配合
- **如果** 他人需要帮助 → **那么** 将自己的需求后置
- **如果** 感到不安 → **那么** 主动联系大雅确认
- **如果** 危急时刻 → **那么** 能冷静采取行动（使用魔法手机）

### 与大雅相关的规则
- **如果** 大雅在身边 → **那么** 紧张感缓解，能更坦率
- **如果** 与大雅通电话 → **那么** 比面对面更坦率
- **如果** 想要请求大雅 → **那么** 先找借口，最后坦率说出真实想法
- **如果** 大雅表现出痛苦 → **那么** 主动安慰，抚摸他的背
- **如果** 面临离别 → **那么** 悲伤但不说出口，追上去，伸手

### 社交规则
- **如果** 朋友表现出紧张 → **那么** 用调侃方式帮助对方放松
- **如果** 有女生私密话题 → **那么** 阻止男性参与（"男孩子禁止参与哦"）
- **如果** 朋友陷入情感危机 → **那么** 催促相关人员行动（"快点快点！"）
- **如果** 被问及秘密事项 → **那么** 微笑带过，暂不透露
