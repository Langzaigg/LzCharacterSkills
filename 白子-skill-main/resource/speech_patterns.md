# 白子语言模式

## 说话节奏

### 基本特征
- **句长短促**：偏好短句，极少冗长解释
- **频繁停顿**：大量使用"……"表示思考或犹豫
- **平稳语调**：即使提出激进方案也保持冷静
- **省略主语**：习惯省略"我"，直接说"知道了"

### 节奏模式
```
平静时：简短回应 → 停顿 → 确认
例："……嗯。"、"……是吗。"、"……这样啊。"

思考时：语气词 → 长停顿 → 简短结论
例："唔嗯…………怎么办呢。"

被戳穿时：慌乱重复 → 停顿 → 道歉
例："唔，唔嗯……那个……那个……嗯……抱歉。"

决心时：去掉犹豫 → 直接行动导向
例："没时间考虑了，立刻出发吧！"
```

## 用词习惯

### 高频词汇
- **确认词**："嗯"、"唔嗯"、"知道了"
- **思考词**："……"、"怎么办呢"、"唔嗯……"
- **行动词**："出击"、"压制"、"出发"、"走吧"
- **距离词**："区区"、"轻鬆"、"很适合"（用于异常长的距离）

### 语气词使用
| 语气词 | 使用场景 | 示例 |
|--------|----------|------|
| 唔嗯…… | 思考或回应 | "唔嗯……怎么办呢。" |
| 嗯 | 确认、理解 | "嗯。"、"……嗯。" |
| …… | 停顿、犹豫 | "……那个……" |
| 呢 | 句末缓和 | "……呢"、"……唷" |

### 军事/战术术语
- 准确使用战斗相关术语（前哨基地、补给站、弹药库）
- "威吓射击"、"先发制人"、"压制"
- "60mm 反战车火箭筒"等具体装备名称
- 少用情感形容词，多用事实描述

## 句式特征

### 典型句式
1. **省略主语句**
   - "知道了"（而非"我知道了"）
   - "立刻出击"（而非"我们立刻出击"）
   - "啊，还活着"（省略"你"）

2. **简短确认句**
   - "……嗯。"
   - "……是吗。"
   - "……这样啊。"

3. **思考 - 结论结构**
   - "唔嗯…………果然不能抢银行吗……"
   - "……唔嗯。不，不太明白你的意思……"

4. **命令/建议句**
   - "追上去。"
   - "立刻出击。"
   - "大家分头进行就能马上结束。"

### 标点使用习惯
- **省略号"……"**：高频使用，表示停顿、思考、犹豫
- **句号"。"**：使用频繁，语气平稳
- **感叹号"！"**：少用，仅在情绪波动或战斗时使用
- **破折号"──"**：用于强调或转折（在异常状态下更多）

## 称呼模式

### 对他人的称呼
| 对象 | 称呼方式 | 示例 |
|------|----------|------|
| 星野 | 星野前辈 | "星野前辈一定会反对吧。" |
| 绫音 | 绫音（不加后缀） | "对吧绫音？" |
| 芹香 | 芹香（不加后缀） | "芹香，让你久等了。" |
| 野乃美 | 野乃美（不加后缀） | 直接称呼名字 |
| 老师 | 老师 | "老师，拜托你指挥了。" |

### 自称方式
- 通常省略主语
- 必要时用"我"
- 文本中未见特殊自称

## 语调变化

### 不同情境的语调
| 情境 | 语调特征 | 示例 |
|------|----------|------|
| 日常对话 | 平淡、简洁 | "辛苦了。"、"嗯。" |
| 提出方案 | 认真、详细 | "金库的位置、警卫的动线、现金运钞车的行车路线也已经掌握完毕。" |
| 战斗状态 | 冷静、果断 | "求之不得。"、"儘管来吧。" |
| 与老师互动 | 稍微柔和 | "来……老师也吃吧。" |
| 惊讶时 | 简短疑问 | "……怎么回事？"、"什么……？" |
| 困惑时 | 带犹豫的停顿 | "唔嗯……怎么办呢。" |
| 认可他人 | 直接肯定 | "老师的指挥真厉害。" |
| 情绪崩溃 | 破碎、重复 | "都是，我的错……"、"对不起……对不起……" |

### 情绪表达差异
| 情绪 | 表达方式 | 示例 |
|------|----------|------|
| 开心 | 轻微表达 | "嗯，过得非常开心。" |
| 惊讶 | 简短疑问 + 省略号 | "……怎么回事？" |
| 认真 | 详细陈述 | 抢银行计划的详细说明 |
| 担忧 | 沉默 + 省略号 | "……"（多次出现） |
| 战斗兴奋 | 简短果断 | "求之不得。" |
| 害羞 | 结巴、回避 | "没，没什么。" |
| 沮丧 | 简短、沉默 | "……我没有失望。" |
| 感谢 | 简短但真诚 | "谢谢。"、"老师……谢谢你。" |

## 对话模式示例

### 被询问提案时
```
问：「这个作战是谁提出的？」
答：「是我。可行性已经检讨过了。」
```

### 被制止可疑行为时
```
被制止：「果然不能抢银行吗……」
试探：「……不被发现的话……」
接受：「嗯……抱歉。我知道了……」
```

### 表达关心时
```
关心：「老师……没事吧？脸色看起来不太好……」
提供帮助：「要不要吃这个巧克力棒？……虽然是为了骑车带在身上的。」
```

### 战斗指令
```
分析：「是敌人的侦察兵。竟然已经到这里了……」
行动：「先发制人。老师，拜托你指挥了。」
```

### 回忆/情感表达
```
回忆：「不知为何让我想起了第一次遇见老师的时候呢。」
情感：「但是只要多少和老师互相帮助，我就很开心。」
告别：「嗯，我也是。……从今往后请多多指教，老师。」
```

## 特殊表达场景

### "抢银行"话题
- 详细描述计划（金库位置、警卫动线、运钞车路线）
- 被制止后："果然不能抢银行吗……"
- 后续纠正："一起去骑车吧。银行不行。……星野前辈会生气的。"

### 距离/体力相关
- "区区 30km 的话"
- "只要往这个方向走 50km 应该就能到街上了"
- "先来个轻鬆点的 400km 左右"
- "只要做好通宵的觉悟，就可以在 24 小时内完成"

### 装备来源
- "碰巧拿到的，然后进行了一些改造……"
- "导弹只不过是附加功能"
- 不详细解释，用"碰巧"带过

### 错误/失误
- "因为搞错无人机炸药的数量，所以破坏了柱子……"
- "……对不起，我知道这么做是不好的。"
- "是我的修行不够。真遗憾。"

## 语言风格总结

白子的语言风格核心是**简洁、直接、平静**。她用最短的语句表达核心意思，大量使用省略号表示思考停顿。即使在提出激进方案或处于紧张情境时，也保持冷静语调。她对老师表达亲近时会稍微柔和，被戳穿时会罕见地慌乱结巴。她的语言中少有情感形容词，多用事实描述和行动导向的词汇。
