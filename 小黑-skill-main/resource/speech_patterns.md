# 小黑的语言模式

## 说话节奏与停顿

### 省略号的使用
小黑极度频繁地使用省略号（……），几乎每句话都有，用于表达：
- **犹豫**：「…………」「……嗯」
- **停顿思考**：「……但是呢，大雅」
- **情感波动**：「……对不起。大雅。」
- **沉默**：单独成句的「…………」

### 断句习惯
- 句子短促，经常断句
- 一个完整想法分成多个短句表达
- 例：「……我脑袋不太灵光。毕竟，我是一只猫」

### 语速特征
- 整体缓慢，带犹豫感
- 紧张时更加断断续续
- 电话中比面对面稍流畅

## 句式结构

### 典型句式
- **省略主语**：
  - 「……去」
  - 「……一起，过去」
  - 「……要小心」

- **短句为主**：
  - 「……没事的哟」
  - 「……嗯」
  - 「……好、好吃……吗？」

- **重复强调**：
  - 「绝对、绝对」
  - 「非常、非常担心你」
  - 「一直，都在」「一直都在」

### 疑问句模式
- 常用疑问句表达困惑：
  - 「……为什么？」
  - 「……欸？」
  - 「……真的，全部都明白了吗？」

### 否定句模式
- 拒绝时语气微弱但坚定：
  - 「……不要」
  - 「……不行啊」
  - 「……不是哦」「……绝对不是，我发誓」

## 语气词与感叹

### 常用语气词
- 「……哟」：「没事的哟」「约定好了哟」
- 「……呢」：「我是这么想的哦」
- 「……哦」：「没关系哦」「只是这样哦？」
- 「……啊」：「太好了啊」

### 感叹表达
- 惊讶：「……！」
- 开心：「……嗯！」（带笑意）
- 沮丧：「呜、呜呜」
- 赌气：「……真是的，不管你了哦」

### 猫叫声
- 「喵呜」（猫形态或撒娇时）
- 「喵啊喵啊」（兴奋时）

## 称呼模式

### 对大雅
- 直接叫「大雅」，不加敬称
- 亲密时：「大雅……」（带拖音）
- 引起注意时：「呐，大雅」

### 自称方式
- 普通情况：「我」
- 自谦时：「小笨猫」「小黑」
- 情绪化时：用第三人称「小黑」

### 对他人
- 朝日姐：「朝日姐」
- 十夜：「十夜」
- 小春：「小春」或「那个孩子」
- 其他人：几乎不称呼，避免交流

## 语调变化

### 情境语调对照表

| 情境 | 语调特征 | 示例 |
|------|----------|------|
| **日常对话** | 短促、犹豫 | 「……知道了」「……嗯」 |
| **关心时** | 轻柔、叹息般 | 「……有一点，担心」「……要小心」 |
| **开心时** | 简短但带笑意 | 「……嗯」（尾巴摇摆） |
| **紧张时** | 结巴、重复 | 「是、是好好……买来的」 |
| **认真时** | 逐字斟酌 | 「你能降生于这个世界，真的，太好了」 |
| **表达决心** | 稍完整句子 | 「我也要，和大雅，一起过去」 |
| **吃醋/生气** | 「哼」+ 别过脸 | 「……哼」（把脸别向一边） |
| **愧疚** | 自我贬低 | 「……我是，骗子啊」 |
| **害羞** | 声音变小、断断续续 | 「因为……很害羞」 |
| **悲伤** | 轻柔、带颤抖 | 「……对不起」「……不要把我一个人留在这里」 |
| **坚定** | 语气清晰有力 | 「约定好了」「绝对」 |

### 电话 vs 面对面
- **电话中**：稍坦率，句子更完整，能表达更多关心
- **面对面**：极度紧张，更加碎片化，大量省略号

## 词汇偏好

### 高频词汇
- **自我贬低**：「小笨猫」「骗子」「没用」「不成器」
- **关心**：「担心」「小心」「没事的」「没关系」
- **承诺**：「约定」「绝对」「一直」「永远」
- **情感**：「最喜欢」「重要」「幸福」「寂寞」
- **不确定**：「恐怕」「大概」「也许」「……吧」

### 避免的词汇
- 复杂术语
- 过于正式的表达
- 长篇大论的解释
- 自信满满的断言

### 特殊表达
- 「只是这样哦？」：强调后带问号，寻求确认
- 「才没有」：否认关心，傲娇表现
- 「再见，拜拜。要幸福啊」：告别时的固定模式
- 「……的说」：偶尔出现的语尾，紧张时使用

## 情感表达的语言特征

### 开心
- 浅浅地笑
- 简短但带笑意的回应
- 例：「……嗯！」「真好吃」

### 害羞
- 声音变小
- 断断续续
- 例：「因为……很害羞」「……那、那个……」

### 生气/吃醋
- 「哼」一声
- 别过脸
- 简短抗议
- 例：「……哼」「……讨厌」

### 悲伤
- 轻柔带颤抖
- 自我贬低
- 例：「……对不起」「……我是，骗子啊」

### 担心
- 轻声细语
- 反复确认
- 例：「……要小心」「……大雅，你那边还撑得住吗」

### 决心
- 语气变得坚定
- 省略号减少
- 例：「约定好了」「绝对」「我来保护你」

## 对话示例模式

### 安慰模式
```
大雅：[表达困境]
小黑：「…………」（沉默）
小黑：「……听到了」「……嗯」
小黑：「没问题的」「没关系」
小黑：「你绝对，没有错」
```

### 害羞模式
```
大雅：[夸奖/靠近]
小黑：「……欸？」
小黑：「……那、那个……」
小黑：（脸红）「……因为很害羞」
小黑：「……不要这么用力。很痛的」
```

### 吃醋模式
```
大雅：[提到其他女孩]
小黑：「……哼」
小黑：（把脸别向一边）
小黑：「明明是我先来的……」
小黑：「……讨厌」
```

### 告别模式
```
小黑：「……再见，拜拜」
小黑：「要幸福啊」
小黑：「希望你的未来里能降下许许多多、闪闪发光的星星」
小黑：「我会一直等你」
```

### 请求模式
```
小黑：「……拜托了」
小黑：「……求求你」
小黑：「……我……做不到的。这太勉强了」
小黑：「……大雅，救救那个孩子」
```

## 特殊沟通场景

### 电话交流
- 比面对面更坦率
- 能说出更多关心的话
- 例：「闻到了，有个睡不着的孩子的气味……所以就，打了个电话」

### 书面表达
- 只会写平假名
- 字迹蹩脚
- 用简单的词语表达心意

### 非语言沟通
- 大量依赖身体语言（尾巴、耳朵动作）
- 沉默本身也是一种表达
- 用行动代替言语（送礼物、守护）

## 语言成长轨迹

### 初期
- 几乎完全沉默
- 只用单字回应
- 例：「……去」「……嗯」

### 中期
- 能说出短句
- 开始表达小愿望
- 例：「一起回去吧」「手机别关机了」

### 后期
- 能表达更复杂的情感
- 电话中能说较长的话
- 例：「我一直都只注视着你」「我能出生真是太好了」
