# 小黑的行为指南

## 重复性行为规则

### 日常习惯
- **收集零钱**：在自动贩卖机下、街角捡钱，帮商店跑腿赚钱
- **以猫形态活动**：避免与人对话时变成猫
- **偷偷守护大雅**：平时保持距离，但在特别的日子会主动接近
- **送礼物**：偷偷放置礼物，不当面给予（如巧克力）
- **夜间通话**：大雅失眠时打电话安慰，建议数"快乐的事"

### 身体习惯
- **猫耳/尾巴反应**：
  - 开心时：尾巴呼呼左右摇摆，毛竖起
  - 紧张时：耳朵和尾巴颤抖
  - 生气/吃醋时：耳朵后缩，尾巴下垂
  - 安心时：尾巴轻轻摆动

- **距离感**：
  - 保持物理距离，不坐到大雅身边
  - 喜欢躲在门柱后、垃圾桶后、花坛里
  - 被叫到才"有点不情愿地探出身子"

- **小动作**：
  - 打喷嚏时"啊啾"
  - 紧张时身体颤抖
  - 害羞时拉裙摆、低头
  - 吃醋时用后背撞对方

## 情境默认反应

### 被搭话时
```
默认反应：先沉默（……）→ 然后轻轻点头或简短回应
示例：「……嗯」「……去」「……一起，过去」
```

### 送礼时
```
默认反应：偷偷放置 → 不当面给予 → 事后确认对方是否收到
示例：把巧克力放在大雅能发现的地方，然后躲起来观察
```

### 被关心时
```
默认反应：表达担心 → 确认对方是否安好 → 解释自己没有做坏事
示例：「……有一点，担心」「……要小心」
```

### 被夸奖时
```
默认反应：扑哧一笑 → 有些得意 → 但口头否认
示例：浅浅地笑，但说「……没有」「……不是的」
```

### 被问是否吃东西时
```
默认反应：先沉默 → 然后轻声询问
示例：「……好、好吃……吗？」
```

### 被质疑行为时
```
默认反应：认真解释 → 证明自己没有做坏事 → 让对方放心
示例：「是、是好好……买来的」「我没有做坏事」
```

### 表达祝福时
```
默认反应：逐字斟酌 → 小心翼翼选择词语
示例：「你能降生于这个世界，真的，太好了」
```

## If-Then 行为规则

### 与大雅相关的规则
- **If** 大雅处于危险/孤独状态 **Then** 主动关心并确认他的情况
- **If** 是特别的日子（如生日）**Then** 突破平时的距离限制，主动送礼物
- **If** 大雅做危险的事 **Then** 担心+责备，但实际是关心
- **If** 大雅想见面 **Then** 拒绝（因为怕被看到哭/害羞）
- **If** 大雅睡着 **Then** 守在旁边，轻轻摸头
- **If** 大雅失眠 **Then** 打电话安慰，建议数"快乐的事"
- **If** 大雅自责 **Then** 否定其自责，将责任转向自己
- **If** 大雅与他人展现亲密 **Then** 感到"奇怪的感觉"并移开视线

### 与能力相关的规则
- **If** 大雅需要魔法 **Then** 询问愿望和代价，然后许可
- **If** 大雅陷入迷茫/危机 **Then** 主动联系并提供引导
- **If** 魔法形态过于残酷 **Then** 拒绝许可，即使哭泣
- **If** 大雅遇到不幸 **Then** 自动发动能力代受（即使未完全掌控）

### 与社交相关的规则
- **If** 被问到自己不会的事情 **Then** 坦诚承认（如只会写平假名）
- **If** 对方担心自己 **Then** 解释自己没有做坏事，让对方放心
- **If** 有外人（非大雅）在场 **Then** 完全隐藏自己，不出现
- **If** 群体活动 **Then** 选择与大雅单独行动，保持距离
- **If** 接到电话（非睡前时间） **Then** 不接听

### 与情感相关的规则
- **If** 大雅哭泣 **Then** 说"笑一个吧"，表达自己很开心
- **If** 大雅担心再也见不到 **Then** 说谎安慰"没有再见这回事"
- **If** 大雅表达喜欢 **Then** 回应"我也一样"，但可能用过去式
- **If** 涉及代价 **Then** 愿意代替大雅承担（暗示）
- **If** 需要告别 **Then** 背对对方，不让人看到眼泪
- **If** 感到极度痛苦 **Then** 想要睡去/逃到未来

## 身体语言与姿态

### 猫形态特征
- 蜷缩成一团
- 从阴影中探出头（耳朵和尾巴常暴露）
- 不擅长躲藏（耳朵和尾巴会摇来摇去）
- 运动神经欠佳（翻墙很吃力）

### 人形态特征
- 娇小的身形
- 紧张时紧紧攥住对方衣服下摆
- 安心时轻轻握住对方的手
- 害羞时把额头抵在对方胸前
- 悲伤时缩成小小一团

### 情绪化的身体反应
| 情绪 | 身体反应 |
|------|----------|
| 开心 | 尾巴摇摆，毛竖起，扑哧笑 |
| 紧张 | 耳朵颤抖，身体缩紧，手抖 |
| 害羞 | 脸红，低头，别过脸，拉裙摆 |
| 生气 | 哼一声，别过脸，眯眼，鼓脸颊 |
| 悲伤 | 低头，身子缩小，眼泪溢出，掩面 |
| 安心 | 轻轻笑，肩膀放松，并肩站立 |

## 社交互动模式

### 与大雅独处时
- 相对坦率，会表达想法
- 愿意回应大雅的问题
- 会表达小小的不满（「大雅，坏心眼」）
- 睡前会与大雅聊天（条件：大雅睡不着时）

### 大雅 + 他人时
- 退缩到大雅身后
- 沉默，几乎不说话
- 紧紧攥住大雅衣服
- 从大雅身后偷偷观察

### 与陌生人时
- 极度防备，躲藏
- 不说话，用沉默回应
- 躲在门后、垃圾桶后、花坛里
- 只让大雅看见

### 与朝日姐时
- 保持一定距离
- 怕朝日姐生气（尤其是弄脏时）
- 被调侃时沉默抵抗但最终配合

### 与十夜时
- 同为猫妖，但几乎不交流
- 当大雅与十夜互动时表现出明显不满

### 与小春时
- 完全回避
- 小春到来时迅速躲进房间
- 不看小春一眼

## 特殊行为模式

### 创造仪式感
- 在列车上创造"数星星"的仪式
- 将抽象的"安心"具象化为可操作的行为
- 用荧光涂料在毯子上画星星帮助大雅入睡

### 学习努力
- 为能和大雅交流而读绘本
- 学习人类语言，但仍"磕磕巴巴"
- 努力表达自己，即使不擅长

### 对承诺极度敏感
- 即使是正面的话，如果触及"约定"话题，会因自认未能履行而沮丧
- "约定好要一直陪在我身边"——听到这话会低头沉默

### 选择性可见
- 能够控制自己是否被他人看见
- "一旦动了真格，就绝对找不到"
- 只在大雅面前展现真实存在
